# To-do List com React + Recoil

Aplicação de gerenciamento de tarefas desenvolvida com React e Recoil.

## Funcionalidades

- Adicionar novas tarefas
- Listar tarefas
- Marcar tarefa como concluída ou pendente
- Remover tarefa
- Filtrar por:
  - Todas
  - Concluídas
  - Pendentes

## Estrutura

- `src/components`: componentes da interface
- `src/atoms`: átomos do Recoil
- `src/selectors`: seletores do Recoil

## Como executar

1. Instale dependências:

	```bash
	npm install
	```

2. Rode em modo desenvolvimento:

	```bash
	npm run dev
	```

3. Build de produção:

	```bash
	npm run build
	```
